Improving Developer Workflow

UX Director · Silicon Labs · 2025
Problem

Developers working across Simplicity Studio and VS Code experienced fragmented workflows, requiring constant context switching and slowing down development tasks.

Approach

I conducted interviews with FAEs to identify workflow bottlenecks, then designed a unified interaction model that reduced fragmentation and clarified task flow across tools.

Impact

Improved task clarity and reduced friction across core workflows, enabling faster onboarding and more efficient debugging for developers.
